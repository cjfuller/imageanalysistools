ImageAnalysisTools is a java library developed for processing and quantifying biological microscopy images. We use it particularly for the identification and quantification of centromeres and kinetochores and provide prebuilt tools for this purpose, but the underlying infrastructure can be used for building tools to process any type of image.

The library has two major goals:

  1. Providing a high-level interface that facilitates rapid development of application-specific image analysis tools: Common image processing needs like visiting all pixels in an image or applying common filters can be accomplished with very little code using a simple interface. Analysis can be scripted using ruby. Complete javadoc for the interface is provided at every release.
  2. Record keeping: analysis methods run with the library produce an output metadata file that keeps a record of exactly what analysis was run (including code versioning information), and what inputs and outputs are associated with the analysis. This record is both human- and machine-readable, and can be re-executed by the library providing a guarantee that precisely the same analysis can be systematically reproduced.
